# MiningServerPlugin

A plugin for a future-themed Minecraft Mining Server

Minecraft Server IP: `futuremines.minekeep.gg`

Discord Invite Link: `discord.gg/8ce6N8QXpm`

### Features
- Vanish command (`/vanish`)
- Combat manager (Disabling `/spawn`, `fly`, etc. when in combat)
- Custom tag page (Integrating with LuckPerms API)
- Custom name tags (Integrating with LuckPerms API)
- Custom portals (Teleports you through the world)
- Autocompress command (`/autocompress`; Automatically compress your items)
- Custom vouchers (`/voucher create <rank>`; Use with right click)
- Discord bridge (synchronizing chats)
- Custom info page (`/info`; View information about the plugin)
- Custom rules (`/rules`; View the rules)
- Discord link invitation (`/discord`; View the discord invitation link)
- God command (`/god`; Invulnerability)
- Invsee command (`/invsee`; View the inventory of another player)
- Fly command (`/fly`; Enable fly mode like in Creative)
- Spawn command (`/spawn`; Go back to spawn)
- Link command (`/link`; Link with discord account)
- Unlink command (`/unlink`; Unlink from discord account)
- Warp command (`/warp <location>`; Warp anywhere)
- Show command (`/show <field>`; Broadcast your hand, offhand, chestplate, etc.)
- Custom join & quit messages
- Custom telepathy effect on all blocks
- WorldGuard integration: Prevent Elytras in PVP, except when being in the PVP region (`/warp pvp`)
- Custom enderchest command (`/enderchest`; Open your enderchest remotely)

### Dependencies
- LuckPerms (Minecraft Plugin)
- JDA (Java Discord API)
